Welcome to Rapyd Library for Code Igniter

Rapyd is a set of components/classes that can help you to develop web applications "rapidly".

Please read the Introduction section of the Rapyd Guide to learn the broad concepts behind Rapyd Library, then read the Getting Started page.

Note: Rapyd is not an "official" or "approved" addon of CI. It's just a project coming from CI fans/community guys.

Who is Rapyd Library For?

Rapyd Library is right for you if:

  • You want a concise mode, component based, to build CRUD's in few line of code.
  • You want a simple way, to build Data GUI in few line of code.
  • You need decent performance, but what you really need is a standardized way to develop your backends.
  • You need broad compatibility with standard hosting accounts that run a variety of PHP versions and configurations.
  • You want a library that requires nearly zero configuration.
  • You want a way to build your applications that does not require you to use the command line.
  • You want a way to build your applications that does not require you to adhere to restrictive coding rules.
  • You think that PEAR is a devil.
  • You are a Designer, so.. "frontend with CI and javascript FX, and Backend with Rapyd".
  • You are a Programmer, so.. "design sucks, I need it works".
  • You are a Designer&Programmer, so.. "I'll make my own Rapyd Theme, then I'll use it for Frontend and to develop the Backend"

Rapyd Features

  • data set: a convenient way to paginate your data.
  • data grid: paged, sortable, customizable, joinable with other components
  • data table: array or active record driven, paged, sortable, customizable, joinable with other components
  • form fields: from a simple input to a complex wysiwyg replacement for textarea
  • data form: a convenient way to manage a form, actions, with a lot of predefined fields, integrated validation and callbacks
  • data filter: a search functionality from 3 lines of code, joinable with grids and sets
  • data edit: a complete crud in few lines, predefined actions, joinable with grids and sets
  • data object: an active record "alternative" to CI a.r., a tiny dynamic ORM (DataObject), with a basic support for relationships

Getting Started With Rapyd

First of all, you need to learn Code Igniter by his guide, trying it etc...

Then you can read the rapyd overview and decide if Rapyd Library can help you to develop CI applications.

The second step is to install Rapyd Library, then read all the topics in the Rapyd Guide (this document). It's tiny and easy.
If you are "experienced" CI programmer, you can probably jump to General Concepts

Once you understand the basics you'll be ready to test rapyd by his Samples Controller.

Next, you can use this documentation as a class reference.

Feel free to take advantage of our Community Forums and remember that Rapyd Library has a Development Website (Trac on CIforge.com), and last version of software can be taken via SVN.
